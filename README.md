# Getting Started

## Env Setup
### Dependencies
* [homebrew](https://brew.sh/) is the easy way
* git
  * `brew install git`
* Python 3.10 
  * `brew install python@3.10`
* virtual environment
  * `python3 -m pip3 install virtualenv`
### Setup
* `git clone `
* `mkdir ~/venv`
* `python3 -m venv ~/venv/spreadsheet-reader-py`
* `source ~/venv/spreadsheet-reader-py/bin/activate`
